# 简介
## ls for windows
### 你是否还在为在windows上没有ls命令而烦恼？
### 难用的dir命令和糟糕的排版
### ls for windows 可以解决你的难题

# 安装
[下载地址](https://github.com/Tianze-ya/ls-for-windows/releases)

将ls.exe放入喜欢的位置并添加环境变量
在命令行中输入`ls`即可使用
在config.txt中配置默认的ls选项

# 功能
### ls
显示当前目录下的文件和文件夹
### ls -h
显示帮助信息
### ls -v
检查版本
